## about OMX-서버모터

OMX-서버모터는 로봇의 관절 구동을 위해 사용되는 고성능 액추에이터로, 정밀한 위치 제어와 안정적인 토크 출력을 제공하는 것이 특징이다.  
본 프로젝트에서는 로봇의 움직임을 구현하고, 모방학습을 통해 학습된 동작을 실제로 수행하기 위해 OMX-서버모터를 활용한다.

OMX-서버모터의 주요 특징은 다음과 같다:

1. **정밀한 위치 제어**  
   엔코더 기반의 피드백 제어를 통해 각 관절의 위치를 정확하게 제어할 수 있다.

2. **강력한 토크 출력**  
   로봇 팔이나 이동 로봇과 같은 시스템에서 충분한 힘을 제공하여 다양한 작업을 수행할 수 있다.

3. **통신 기반 제어**  
   시리얼(UART) 또는 기타 통신 프로토콜을 통해 외부 제어기(예: Jetson Nano, PC)와 연결되어 명령을 주고받는다.

4. **확장성**  
   여러 개의 모터를 동시에 제어할 수 있어 다관절 로봇 시스템 구성에 적합하다.

본 시스템에서는 OMX-서버모터를 통해 수집된 동작 데이터를 기반으로 모방학습을 수행하며, 학습된 결과를 다시 모터 제어에 적용하여 로봇이 사람의 동작을 자연스럽게 재현할 수 있도록 한다.

이를 통해 정밀한 로봇 제어와 인공지능 기반 동작 학습을 결합한 통합 시스템을 구현할 수 있다.
